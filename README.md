# Git Checkout Branch

Git Checkout Branch (gcb) is a gem for quickly checking out gems based on partial names.

## Installation

    $ gem install gcb

## Usage

```
$ gcb weird
Switched to branch 'my-weird-branch-1337123456'
$ gcb ast
Switched to branch 'master'
```
